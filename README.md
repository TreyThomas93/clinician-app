# Clinician App

A demo app that allows a user to log in and view fake patient records.

## Dependencies

> dependencies:

- google_fonts: ^3.0.1
- flutter_riverpod: ^2.0.0
- grouped_list: ^5.1.2
- package_info_plus: ^1.4.3+1

> versions:

- flutter: 3.3.9
- dart: 2.18.5
