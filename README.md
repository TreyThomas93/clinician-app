# Clinician App

A demo app that allows a user to log in and view fake patient records.

## Dependencies

> dependencies:

- flutter:
- google_fonts: ^6.2.1
- flutter_riverpod: ^2.5.1
- grouped_list: ^5.1.2
- package_info_plus: ^8.0.2

> dev_dependencies:

- flutter_tests
- flutter_lints: ^3.0.0
- custom_lint: ^0.6.7
- riverpod_lint: ^2.3.13
